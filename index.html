教学指导

通过此交互工具，学生可以直观理解二次函数顶点式的几何意义：

  • 参数 a 决定抛物线的开口方向和大小（|a|越大开口越窄）
  • 参数 h 决定顶点的水平位置（x坐标）
  • 参数 k 决定顶点的垂直位置（y坐标）
  • 对称轴始终为垂直线 x = h
  • 顶点坐标直接由 (h, k) 给出

顶点式与一般式的转换

一般式：y = ax² + bx + c
顶点式：y = a(x - h)² + k
转换公式：
h = -b/(2a)
k = (4ac - b²)/(4a)
顶点坐标：(h, k) = (-b/(2a), (4ac - b²)/(4a))
